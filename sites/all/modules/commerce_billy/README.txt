Commerce Billy
**************
Lightweight approach for a billing system for Drupal Commerce. It uses the
commerce order entity to hold invoice information, the order number is the
invoice number.
Billy = Billing Light


Administration
**************
Go to: Store > Configuration > Billy invoice settings
       /admin/commerce/config/billy-invoice
